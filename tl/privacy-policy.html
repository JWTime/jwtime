Patakaran sa Privacy

Paano pinamamahalaan ng JW Time ang lokal na data, mga koneksyon at diagnostics

Huling update: Hunyo 20, 2026

1. Panimula

Ang JW Time ay isang independiyente, libre at hindi komersiyal na proyekto. Ipinapaliwanag ng patakarang ito kung anong data ang lokal na sine-save, anong mga koneksyon sa Internet ang maaaring gawin, at anong mga kontrol ang nasa user.

Hindi nangongolekta o awtomatikong nagpapadala ang JW Time sa developer ng personal na data, telemetry, ulat ng paggamit, crash report o log.

2. Data na Lokal na Sine-save

Maaaring i-save ng JW Time sa computer ang mga setting ng app, mga profile ng kongregasyon, mga programa, imported na file, media cache, cache ng programa, audio recording kung pinagana, backup at teknikal na log.

Ginagamit ang data na ito para gumana ang app, mapanatili ang mga kagustuhan, makapagtrabaho offline at matulungan ang user na mag-diagnose ng mga lokal na problema.

3. Pagbabahagi ng Data Ayon Lamang sa Iyong Pinili

Hindi awtomatikong nagpapadala ang JW Time sa developer ng personal na data, estadistika ng paggamit, telemetry, crash report, log o mga resulta ng test ng koneksyon.

Kung hihingi ka ng tulong, ikaw ang magpapasiya kung kokopyahin at manu-manong ipapadala ang mga log, screenshot o diagnostic na resulta. Sa kasong iyon, ang ibinahaging data ay gagamitin lamang para magbigay ng suporta.

4. Mga Koneksyon sa Internet

Maaaring kumonekta ang JW Time sa opisyal na mga source para mag-download o mag-update ng mga programa at pampublikong media, sa website ng JW Time para sa dokumentasyon, mga komunikasyon at mga file ng suporta, at sa mga panlabas na serbisyo para lamang sa hayagang mga function gaya ng network diagnostics o pansamantalang remote na access.

Ginagamit ang mga request na ito para kunin ang nilalaman o suriin ang connectivity. Hindi ito telemetry na ipinapadala sa developer.

5. Lokal na Web Server at Remote na Access

Kapag pinagana, ginagawang available ng Web Server ng JW Time ang mga function para sa monitoring at kontrol sa lokal na network. Makikita ng mga device sa parehong LAN ang operational data gaya ng pamagat ng bahagi, natitirang oras at status ng timer.

Kung pinagana ng user ang pansamantalang remote na access, dumadaan ang traffic sa panlabas na serbisyong kailangan para gawin ang koneksyon. Kinokontrol ng user ang pag-activate at ang mga available na proteksyon, gaya ng PIN o mga role.

6. Pag-iimbak at Pagbura

Nananatili ang data sa computer ng user. Awtomatikong nililinis ang ilang pansamantalang file, cache o lumang package kung nakatakda sa app; ang iba ay maaaring manu-manong burahin mula sa mga setting o sa pamamagitan ng pagbura sa mga lokal na folder ng app.

Ang mga lokal na file na manu-manong idinagdag ay nananatili sa lokasyong pinili ng user at hindi inaalis ng JW Time nang walang hayagang aksyon.

7. Mga Karapatan ng User

Dahil hindi nag-iingat ang JW Time ng remote archive ng data ng user, maaari mong pamahalaan, itama o burahin ang iyong data direkta sa iyong computer: mula sa mga setting ng app, sa mga lokal na folder, o sa pag-uninstall ng app at pag-alis ng lokal na data.

Para sa praktikal na impormasyon tungkol sa nilalaman, media at cache, tingnan din ang FAQ.

Para sa tulong, maaari kang makipag-ugnayan sa amin.
