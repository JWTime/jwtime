FAQ

Mabilis na mga sagot tungkol sa privacy, nilalaman, cache at mga koneksyon

Kalagayan ng proyekto

Opisyal bang app ang JW Time?

Hindi. Ang JW Time ay isang independiyente, libre at hindi komersiyal na proyektong nilikha ng mga boluntaryo. Hindi ito kaanib, ini-sponsor, o opisyal na inaprubahan ng anumang organisasyon.

Komersiyal ba ang JW Time?

Hindi. Ibinabahagi nang libre ang JW Time at wala itong advertising, pagbili sa loob ng app, o bayad na kakayahan.

Nilalaman at media

Sinusunod ba ng JW Time ang opisyal na mga tagubilin?

Dinisenyo ang JW Time para sa lokal, personal, libre at hindi komersiyal na paggamit, habang sinusunod ang opisyal na mga tagubilin sa paggamit ng nilalaman. Hindi kasama sa installer ng app ang opisyal na nilalaman at hindi ito muling ipinamamahagi sa pamamagitan ng sarili nitong mga server.

Paano kinukuha ang media?

Kapag ginagamit mo ang Media window, maaaring hilingin ng JW Time ang mga file sa pamamagitan ng opisyal na channel na pampublikong available at i-save ang mga ito sa lokal na cache. Bilang alternatibo, maaari kang magdagdag ng mga lokal na file na nasa computer na.

Privacy at mga koneksyon

Nangongolekta ba ang JW Time ng personal na data o telemetry?

Hindi. Hindi nangongolekta o awtomatikong nagpapadala ang JW Time sa developer ng personal na data, telemetry, ulat ng paggamit, crash report o log. Ang mga setting, profile, programa, recording, log at cache ay nananatili sa computer ng user.

Nagpapadala ba ng data sa developer ang test ng koneksyon?

Hindi. Ipinapakita ng test ng koneksyon ang resulta sa app. Maaari mo itong kopyahin sa clipboard at manu-manong ibahagi lamang kung gusto mo ng tulong.

Anong mga koneksyon sa Internet ang maaaring gawin ng JW Time?

Maaaring kumonekta ang JW Time sa opisyal na mga source para sa mga programa at media, sa website ng JW Time para sa gabay, mga komunikasyon at support file, at sa mga panlabas na serbisyo para lamang sa hayagang mga function gaya ng network diagnostics o pansamantalang remote na access, kung pinagana ng user.

Mga lokal na file at cache

Nananatili ba magpakailanman ang na-download na mga file?

Hindi. Ginagamit ang na-download na mga file bilang lokal na cache o offline na mga package. Awtomatikong nililinis ang ilang pansamantala o lumang file kung iyon ang itinakda; maaari mo ring i-clear ang cache, mga package at offline na media mula sa mga setting ng app. Ang mga lokal na file na manu-manong idinagdag ay nananatili sa lokasyong pinili ng user.

Magagamit ko ba ang JW Time offline?

Oo, para sa nilalamang na-download na, na-import, o manu-manong idinagdag. Kailangan ng koneksyon sa Internet para sa unang download at mga update ng nilalaman.
