First start wizard

At the first start JW Time shows a wizard that collects the essential settings before opening the app. This is the recommended path for a new installation, because it avoids having to search for each option right away in the settings panels.

  • Language and format: choose interface language, program language and time format.
  • Congregation: Set the name of the initial profile, useful if you manage multiple profiles in the future.
  • Meetings: indicates days and start times for the automatic countdown.
  • Meeting options: activate, if necessary, adaptation of durations and automatic pause between parties.
  • Screens: Configure Time Counter Screen and Public Screen on the correct monitors.

At the end of the wizard you can open JW Time and complete only the more advanced options, such as Media, Web Server, Recordings and Backups.

Quick check if you skip the wizard ↑ Table of Contents

If you close or skip the wizard, you can manually configure the essentials from Settings and the main window. This checklist also serves as a quick check before the meeting.

  1. Language, profile and program

    Go to section 9 · General for language and time format, then to section 10 · Congregations if you need to create or select a profile. Check that loaded schedule corresponds to the week and the desired language.

  2. Time Counter Screen and Public Screen

    In the section 12 · Screen select the correct monitors and activate the windows you will use. To change the appearance of the Time Counter Screen see section 5.3 · Customizing the Time Counter Screen.

  3. Media and contents

    Open the section 4 · Media Window to check songs, videos and images of the week. Cache, quality and background options are in section 13 · Media.

  4. Timers, messages and recording

    Use the main window to start timers, send messages to the Time Counter Screen and open the REC panel. If you record meetings, first check the options in section 16 · Recordings.

  5. Web Server and backup

    If you use remote control, QR or browser consultation, configure section 15 · Connection and Server. To work without a network or transfer configurations, check section 17 · Packages and offline backups.

Part I · Interface and Operation

1 · Main Screen ↑ Table of Contents

The main screen is the heart of the application. From here you can start, manage and monitor all timers of the meeting, view key information such as the title of the current part, the clock and the time residue, as well as checking important messages and notifications.

Main areas

1.1 · Part Title and Clock

The title of the part in progress is shown in the top line, while the clock with the current time or, when you start a timer, the countdown of the remaining time. In 12 hour format also appears AM/PM suffix in smaller size to improve readability.

1.2 · Timer Table

Lists the meeting agenda (title, duration, status). Content loads automatically based on the day selected: the app recognizes whether it is Midweek or Weekend and proposes the correct schedule. For the buttons and detailed actions see section 3 · Manipulating timers.

If you enable Show End column in General settings, the table also shows when the current part and each upcoming part would end. It is not a fixed schedule time: it is a live projection based on the current time, the remaining timers and any accumulated timing offset.

1.3 · Estimated End Time

Shows the expected closing time based on the schedule. The deviation appears next to:
+m:ss (yellow within 1', red beyond) = delay
−m:ss (green) = advance
The time and indicator are updated in real time. If the adaptives for Bible Study / Tower of are active Guard, the durations are recalculated to fit within the closing time.

1.4 · Pause/speaker change time

Cumulative counter hh:mm:ss of technical pauses/speaker change: sums the time in which the timer is been paused. During the pause the part timer remains stopped while this counter increases. One click on the indicator it resets the total to zero.

1.5 · Main timer management keys

Controls to start, stop and manage timers: Stop (red), Pause, and Start/Next (green). The Pause button pauses the part timer (the count remains stopped) while the pause/change time speaker continues to increase. To exit the pause, press Pause again or skip to the next part with Start/Next: the accumulated time is recorded in the counter referred to in point 1.4.

1.6 · News (communications)

When there are official communications for the app, a "News" box/alert appears on the Main Screen. Clicking it opens the panel with the update summary.

1.7 · Quick Tools

Quick buttons to open settings, minimize or restore the window, and access help. They are separated from the operating modes to leave the bottom line lighter.

1.8 · Operating modes

From here you can switch to the Media Window, Program management, Messages and the REC panel. The black dot next to REC indicates the recording status.

2 · Program and schedules ↑ Table of Contents

This section illustrates how to load the correct schedule for the week, create a manual schedule when needed and Quickly scroll through previous, current and next weeks.

By default, the app automatically loads the current week's lineup: midweek from Monday to Friday and the weekend one on Saturday and Sunday.

2.1 · Automatic programming

  1. Midweek: loads the weekday schedule.
  2. Weekend: Load up the Saturday/Sunday lineup, asking for the public speaking song.

2.2 · Week selector and calendar preview

The Week button always shows the range (e.g. "Week: 20 – 26 Oct.") with a badge which indicates the offset from the current week (0, +1, -2, etc.). One click opens the compact calendar for jump directly to the desired week without scrolling the Previous/Next buttons several times.

  • Instant badge: understand if you are looking at the current, future or past week.
  • Weekly calendar: choose the weeks directly in a practical way.

2.3 · Visit SC

The Visit SC, located between the Week selector and Load from backup, transforms the schedule when the circuit overseer's visit is present. When pressed, the button lights up (ON) and the app automatically prepares the special parts planned for the visit.

  • Midweek meeting: the Congregation Bible Study is converted into a fixed 30-minute segment entitled "SC Visit".
  • Weekends: A dedicated entry is added after the Watchtower Study and the durations are adjusted to meet the closing time.
  • Dedicated colors and warnings: the Visit SC line adopts its own palette and prepares the specific final song.
  • Quick deactivation: by turning off the toggle, the table immediately returns to the standard schedule without losing the other changes.

2.4 · Load from backup

The Load from backup opens a list of packages saved locally, sorted by date, type of meeting and language. It is ideal for working offline or restoring already prepared programs to another PC.

  • Detailed list: each line shows week start date, type (Midweek/Weekend) and language.
  • Secure Load: Packages are verified and applied directly to the current table.
  • Clear messages: if there are no backups available, the app immediately shows a warning.

2.5 · Manual List Mode

The manual list is useful when the schedule does not come from the weekly program or when you want to prepare a special program. You can create or edit entries via the dedicated editing window, setting the type, title and duration, and save the result to reuse it later.

2.5.1 · Manual Panel

  • Manual: activates the custom schedule and shows the dedicated commands under the program selector.
  • Save manual list: export the current schedule to a file to reuse or transfer to another PC.
  • Load manual list: import a saved list and update the linked table, Time Counter Screen and Media.
  • Clear List: Clears the current week's manual lineup and also removes manual media linked to that draft.

2.5.2 · Create and save

  • Activate manual mode from the Program panel.
  • Use table commands to add, delete, or reorder rows; when needed, JW Time opens the voice editing window.
  • Save the list when you want to keep it for another time.

When you save, JW Time asks you what to include in the .jwtimeplanfile. Choose Manual program + media if you also want to transfer local files, URLs and groups from the Media Window. Choose Only manual time program if you only need the schedule with titles and durations.

2.5.3 · Reload or empty a list

  • Use Load manual list to retrieve a saved schedule.
  • If the list contains lines recognized as standard songs or parts, JW Time applies consistent colors and behavior.
  • Linked windows are updated after loading, so Time Counter Screen and Media stay aligned with the table.
  • Use Clear List when you want to restart from an empty schedule for the same week.

2.5.4 · Linked Media

When you attach media to a manual list, saving and loading maintain the link to the prepared program. This is useful for special meetings, local assemblies, or programs that do not follow the standard week.

If a file is no longer available, the app notifies you when you open or use the Media Window, so you can replace it before the meeting.

To add a song, use the input menu and choose the song item. This way JW Time correctly records the song number, connects the available media and the Time Counter Screen adapts the display without having to manually write the title in the line.

3 · Manipulating timers ↑ Table of Contents

This section explains the main timer management operations — such as adding, editing or deleting entries — and the use of start, jump, and reset commands.

3.1 · Main operations

  1. Line menu: open the three dots on the right for the actions available on the selected part.
  2. From the menu you can start the line or move to the next one, add a line above or below, duplicate it, reset it or delete it.
  3. The Add Line Above and Add Line Below commands open a submenu with Song, Part/Speech and Advice.
  4. Edit Title/Duration: double click on the cell, then confirm.
  5. Formats Duration: 5 → 5:00, 1:30 → 1:30 minutes.
  6. Starting and progressing:
    • Start line starts the selected part.
    • When a line is in progress, the command becomes Move to the next.
    • Stop/Pause manage the running timer from the bottom bar.

3.2 · Real-time updates

The timer updates constantly: the main countdown at the top shows the remaining time of the active part, while the corresponding row remains highlighted in the table. The color of the top bar picks up the color of the part in progress.

If you change a duration or move on to the next part, the Estimated End Time is recalculated immediately. This allows to check the lead/lag delta and the time accumulated for pauses or speaker changes without opening other panels.

3.3 · Automatic Adjustment — Bible Study and Watchtower Study

If Adaptive Bible Study and/or Adaptive Watchtower Study are enabled in Settings, the appropriate app dynamically the duration of the selected part to maintain the closing time of the meeting. Adaptation concerns only the parts with active option and can automatically increase or reduce the minutes based on the accumulated advance/delay. In the image, the affected line is isolated and the adjusted value is highlighted in orange, so it is immediately clear which timer has been corrected. You can always intervene manually: the adaptive recalculates starting from the new situation.

4 · Media Window ↑ Table of Contents

The Media Window collects the contents linked to the week: songs, videos, images and useful references to the part. It can help the operator prepare for the meeting, check available content, and launch it when needed.

Usage note: the Media Window gives the operator a support tool for preparing, checking, and managing multimedia content that is already available. Its practical use depends on local needs and should always be in harmony with congregation direction and arrangements. In case of a technical need, it can also be used as a support or fallback solution.

4.1 · Notice and media sections

Area 1 indicates when the official media of the week is not yet available offline. You can download them immediately or postpone. The same behavior can be automated from the section 13 · Media settings, activating the automatic download of official media. Below the notice you will find the sections of the schedule, with colored title, item counter and quick commands.

Note: downloading official media requires the network to allow JW Time to access jw.org and the resources used for media. The alternate download introduced for programs is not available for media. If routers, firewalls, DNS services, or content filters block religious sites or JW Time access, official media may not download; in that case you can use media already available offline or add local files manually.

4.2 · Contents List

Area 2 contains tabs with preview, title, reference and start button. The colors reflect the areas of the schedule, so you can quickly locate songs, images, videos and references related to the part.

From the context menu you can manually mark an item as used or not used. If Dim used media is enabled, used items remain recognizable with a more subdued style. The official video, audio and image pickers also include Global mode when you need to search outside the current category.

4.3 · Lower bar

Area 3 collects the quick commands of the Media Window: downloaded contents, preview, synchronization, background and dedicated settings. It stays at the bottom to be reachable even when the list is long.

4.4 · Zoom controls

Area 4 collects the Zoom controls of the Media Window. They are designed for those who show media even in video conference and work with the Public Screen on the room monitor.

  • Sharing- Sends the configured shortcut to Zoom to start or stop sharing.
  • Show/Hide Zoom: used to show the speakers or the Zoom window on the public monitor. JW Time temporarily hides the Public Screen and lets you see what is underneath: for this reason the Zoom speakers window, or a duplicate screen with Zoom, must already be positioned behind the Public Screen before pressing the button.
  • Zoom Auto: Enables Zoom automation for the session. When you start a piece of media, JW Time can automatically send the sharing shortcut and, if configured, stop it at the end.

Before using these keys, set up a global screen sharing shortcut in Zoom and enter the same combination in section 14 · Zoom. If the two shortcuts do not match, JW Time sends the command but Zoom does not react as expected.

4.5 · Pre-meeting and background music

The Background panel manages the music before the meeting and, if configured, also after the meeting, without mixing it with the schedule media. During playback it shows current song, next song, progress and duration.

You can start, pause, skip to the previous or next song, activate shuffle and open the playlist. The music settings, including automatic behavior and fades, are configured from the section 13 · Media.

5 · Public Screen and Time Counter Screen ↑ Table of Contents

JW Time 4.0 separates the views: the Public Screen is used for content visible to the room, while the Time Counter Screen remains dedicated to those who follow the times.

5.1 · Public Screen

The Public Screen shows countdown, welcome message, year text, logo and content views in a separate window.

  • Before meeting: Shows the countdown and configured text.
  • During the meeting: can remain available for public content or dedicated scenes.
  • Settings: monitor, message, countdown, logo and layout are adjusted in the section 11 · Meetings and in the section 12 · Screen.

5.2 · Time Counter Screen

The Time Counter Screen shows the clock, countdown of the part in progress, messages for the speaker and states related to time management. It is the most suitable view for podium, chairman or time workers.

  • Display on selected monitor or free window.
  • Current part timer and contextual information.
  • Full or partial screen messages sent from the main window.

Layout customization is covered separately in section 5.3 · Customizing the Time Counter Screen, thus daily use of the screen remains separate from the graphics settings.

5.3 · Customize the Time Counter Screen ↑ Table of Contents

The Time Counter Screen can be customized from the Customize button in the section 9 · General.

You can start from a preset already available or modify the layout according to your needs. The Public Screen has own settings in the section 12 · Screen; here we focus the work on the layout of the Time Counter Screen.

Starting from a preset

A preset is a starting layout.

You can select a preset in the section 9 · General and use it as is, or open the customization editor to edit it.

Choose a scene

The layout is organized into scenes.

Each scene corresponds to a different situation on the Time Counter Screen. This allows you to show a different layout based on the time or type of part being viewed.

Before making changes, always check which scene is selected.

Available Layer Types

When you press + in the Layers panel, JW Time shows the item types you can add to the scene. Some levels read real meeting data and change automatically; others are used to build the graphic layout.

It is not convenient to show all the properties of each level in the manual, because the properties panel changes based on to the selected type. Here you will find what each level does and when to use it.

Main dynamic levels

  • Timer: shows the time of the active part. It can be set as countdown or countup.
  • Song: Show song information when the scene is about a song.
  • Messages: shows messages sent from the main window, both in full screen and in partial format.
  • Clock: Shows the digital time.
  • Analog Clock- Adds a customizable graphical clock.

Meeting information

  • Current part: Shows the title of the current part.
  • Next part: shows the next part, useful for preparing the change.
  • Previous part: shows information about the part just finished, when the layout requires it.
  • Estimated End- Shows the end time calculated by the app.
  • Delay/Early: shows the delta compared to the scheduled time.
  • Pause info: shows information related to pauses or technical time.
  • Progress bar: graphically displays the progress of time.

Free graphic layers

  • Background: adds a colored box, useful for creating bands, contrast areas or visual containers.
  • Text: adds a personalized fixed text.
  • Image: Adds a logo or decorative image.

The Background, Text and Image layers can be added multiple times. For other levels, if an element of the same type already exists, the editor offers you to link it to another scene or to create an independent one when required.

Editing the layout

In the editor you can edit the layers present in the selected scene.

For each layer you can change the position and size in three ways:

  • with the mouse, moving or resizing the box in the preview
  • with the keyboard, using the arrows to move it
  • with the Position and Size values ​​shown under the preview

The X and Y values ​​control the position of the layer.
The Width and Height values ​​control the size of the layer.

All these values ​​are expressed as a percentage.

In the layers panel you can also manage thesuperposition order: every level can stay above or below the others, determining which appears in the foreground in case of overlap. You can change the order by dragging the layers in the list or by using the buttons ▲ and ▼ underneath it.

Move, resize and order layers

After selecting a level, you can:

  • drag it with the mouse to move it in the preview
  • use the keyboard arrows to adjust its position (arrows: 0,5% — Shift: 2,0% — Ctrl: 0,1%)
  • drag the edges or corners of the frame to resize it

Below the preview you will find the numeric controls X, Y, Width and Alt to adjust position and size precisely.

After changes, save the layout and apply it. You can reopen the editor at any time to update it.

6 · Management of messages ↑ Table of Contents

The Messages function allows you to communicate discreetly and clearly with the speaker during the meeting, allowing you to warn him or give him instructions without interrupting or disturbing the progress of the part. Messages appear on the Time Counter Screen, so the speaker sees them in real time without distractions.

6.1 · Full screen message

  1. Click in the area where it says Insert message.
  2. Enter the text.
  3. Press Full screen → the text appears full screen on the Time Counter Screen monitor.

6.2 · Partial screen message

  1. Click in the area where it says Insert message.
  2. Enter the text.
  3. Press Partial screen → the text appears in the Time Counter Screen window without completely covering it, allowing the speaker to see both the message and the timer.

6.3 · Mirror Time Counter Screen

When you send a message, a resizable and moveable preview window automatically appears showing exactly what the speaker sees in the Time Counter Screen monitor. This allows you to check the display in real time of the message and the timer.

7 · REC button and recording panel ↑ Table of Contents

The REC button is located in the bottom bar next to Program and Message. The dot next to the text shows the recording status (wait, active, paused or MP3 conversion), while the REC panel provides quick controls to open the JWTimeREC folder, manage manual recordings, and monitor PC and microphone audio levels.

7.1 · Button indicators and states

  • Waiting: the dot remains gray and the panel shows "Waiting" / "No active recording".
  • Recording in progress: the dot becomes flashing red and the panel shows "Recording in progress" with the part's title and timer.
  • Pause: the dot changes to orange; registration is suspended and can be resumed.
  • MP3 conversion: the dot turns blue and the button remains deactivated until the file export is finished.

7.2 · REC quick controls

The REC panel collects the controls to be used without entering the settings:

  • Open folder: directly opens the JWTimeREC directory containing the recordings.
  • Enable/Disable recording: enable or stop manual recording without entering the settings.
  • Pause/Resume: button visible during recording to temporarily suspend the audio (e.g. prayers) and resume it on the same file.
  • Online details: the title of the active row and the updated duration appear under the status.
  • Automatic messages: immediately indicates when recording is disabled or when a conversion is in progress.

7.3 · Toggle area with audio levels

When you open REC, an always visible panel appears in the lower bar with recording status and audio meter. Here you can start/stop manual recordings, monitor PC/Mic levels and open the JWTimeREC folder with a single click.

8 · Web Monitor & Server ↑ Table of Contents

When the Web Server is active, you can open it from another device on the same local network via browser. This allows the Time Counter Screen to be displayed on external projectors, tablets or computers, and is useful for the chairman who follows the timer of the various parts.

For the complete configuration of the Web Server, remote control, remote link and QR, see section 15 · Connection and Server.

8.1 · View the Web Monitor in the local network

From the browser you can:

  • View timers and the Time Counter Screen in full screen.
  • Print the schedule.
  • Check the timing progress in real time.
  • Perform remote operations on the timer and program, if remote control is enabled.

8.2 · Control timer and program from browser

If remote control is enabled, from the browser you can intervene on timer and program without being in front of the main computer.

Available operations depend on the configured permissions and may include:

  • start the timer
  • pause and resume
  • stop
  • control the program/schedule
  • modify the durations or values available in the schedule, if the authorized role allows it

You can open the Web Server by entering the address shown in the app, or using the QR if available.

Depending on the configuration, different levels of access may be provided, for example for access only viewing or for control and modification. If configured, access may require a PIN.

For the complete configuration, see section 15 · Connection and Server.

8.3 · Remote access for videoconference meetings

When the person who needs to follow or control JW Time is not in the same local network, you can generate a temporary remote link. This scenario is useful for videoconferencing meetings, remote assistance or management from a connected device to another network.

  • Same web interface: the remote link opens the Web Monitor from the browser, as happens in the local network.
  • Temporary link: the remote link changes with each activation and should not be considered permanent.
  • Remote PIN- When configured, protects functions accessible from outside the local network.
  • Use with videoconference: allows anyone following the meeting from outside the network to consult or check the program according to the set permissions.

The operational part is configured in section 15.2 · Remote Web Server. Zoom settings remain separate: Zoom manages sharing and shortcuts, while the remote link concerns the Web Monitor.

Part II · Settings Panels

In this part you can configure the behavior of the app: general settings, congregation profiles, meetings, Time Counter Screen, Media, Zoom, Web & Server Monitor, Recordings and Backup & Restore.

The SAVE button applies all changes.

9 · General ↑ Table of Contents

Appearance and format

  • Select the language: change the interface language.
  • Program languages: defines in which language to download or show the weekly schedule.
  • Select the theme: set the graphic theme of the app.
  • Interface scale: adapt text and control sizes; the change applies after reboot.
  • Time format: alternates 24h format and 12h AM/PM format.

Timer management

  • Auto pause: automatically inserts a pause between one part and the next when you press Start/Next.
  • Show End column: shows in the program table the projected end time for the current and upcoming parts, calculated live from the current time, remaining timers and any accumulated timing offset.

Startup

  • Start with Windows: Opens JW Time automatically when you sign in to Windows.

News

  • Show what's new at startup: opens a concise summary of the features introduced in the app, so those who use JW Time can quickly find out what's worth trying.
  • You can deactivate it when the group is already familiar with the installed version.

10 · Congregations ↑ Table of Contents

This panel allows you to manage congregation profiles. Each profile can maintain dedicated configurations, so JW Time can adapt to different contexts without having to reconfigure windows, media and layout every time.

Congregation Profiles

  • Congregation List: shows the available profiles. Each profile maintains its own language, server, screens, recordings, media and settings.
  • New Congregation- Create a separate profile to configure.
  • Rename: changes the name of the selected profile without losing the settings.
  • Duplicate: creates a copy of the current profile, useful when two configurations are similar.
  • Delete: removes the selected profile when it is no longer needed.

Import / Export

  • Export congregation: save only the selected profile.
  • Import congregation: loads a previously exported profile.
  • Export- Creates a full backup of all congregations.
  • Import- Restores a backup that contains multiple congregations.

11 · Meetings (Meeting Start Timer) ↑ Table of Contents

This panel is used to program the meeting start countdown on the Public Screen. You set the day, time and advance (in minutes) you want to do start the countdown.

11.1 · Scheduling (Day · Time · Advance)

  • Add meeting: inserts a new row in the time table.
  • Day: choose the day of the week on which the meeting is held.
  • Time: set the start time of the meeting.
  • Advance (min): how many minutes before the set time the countdown must appear on the Public Screen.
  • -: delete the time line that is no longer needed.

11.2 · Special Events

  • Add Special Event: Create events such as the circuit overseer's visit, Memorial or manual events.
  • Visit SC: apply the rules and times dedicated to the visit.
  • Memorial- Record a special appointment with a specific date and time.
  • Edit / delete: the buttons on the right allow you to update or remove the event.

11.3 · Adapt Time

  • Adaptive Bible Study- Automatically adjusts the Congregation Bible Study to help meet the end time.
  • Adaptive Watchtower Study: Apply the same logic to the Watchtower Study.

Note: the start times are for the public countdown; the Estimated End Time instead depends on the schedule and any duration adjustments.

12 · Screen ↑ Table of Contents

This panel collects the settings dedicated to the Time Counter Screen and the Public Screen. From here you can choose where to show the information, how to present it and which elements to make visible during the meeting.

  • Enable: activates or deactivates the Time Counter Screen.
  • Select monitor: choose the monitor or window in which to display the Time Counter Screen.
  • Identify: shows a reference on the selected monitor, useful when multiple displays are connected.
  • Mode: choose Full Screen, Free window, or Transparent free window.
  • Layout: select the graphic preset of the Time Counter Screen.
  • Customize: opens the editor of the selected layout.

Public Screen

  • Activate Public Screen: enables the window dedicated to the contents visible to the room.
  • Activate countdown for the public: shows the countdown to the start of the meeting on the Public Screen.
  • Monitor: Select where to open the public window.
  • Layout: chooses the graphic preset of the Public Screen.
  • Show year text: enable or hide the year text.
  • Show JW Logo: Show or hide the logo in the public scene.
  • Year text: allows you to insert custom text; if it remains empty, the year text in the program language is used.

Pointer protection

  • Pointer protection: keeps the mouse pointer out of the Public Screen and Time Counter Screen while they are visible full-screen. Protection operates only while these windows are full-screen.
  • Ctrl+Alt+Shift+M: temporarily suspends protection; press the shortcut again to reactivate it. JW Time briefly shows the current status in the main window.

13 · Media ↑ Table of Contents

This panel manages the behavior of the Media Window, the local cache and the options linked to the contents used during or after the meeting.

Media functions

  • Media button and function: shows or hides the Media button and enables related functions.
  • Dim used media: makes used items easier to recognize in the Media Window.
  • Allow external URLs: after confirmation, allows HTTPS URLs outside jw.org and wol.jw.org and shows them in the integrated browser.

Media Gathering

  • Automatically download the official media: helps make the official media for the week available in advance when the network allows it.
  • Improved management of manual programs and automatic downloads of official media: downloads stay anchored to the current week, and progress for later weeks is shown more clearly.
  • Automatically update songs offline: keeps the catalog of downloaded songs updated.
  • Video quality: choose the resolution of the official videos.
  • Weeks: indicates how many weeks to keep or prepare.
  • Max Space: Limits the space used by the media cache.
  • Folder / delete: Opens the contents folder or cleans downloaded files.

Songs and music

  • Meeting Video Songs: manages download, quality and folder of the songs videos.
  • Songs - Music before/after meeting: manages the music used as background.
  • Random play: Plays songs in random order.
  • Automatically stop at the beginning of the meeting: stops the music when the program starts.
  • Stop music before the meeting starts: defines how many seconds before to stop the background.
  • Fade duration: adjusts the audio fade when closing or changing songs.

14 · Zoom ↑ Table of Contents

This panel contains useful settings when JW Time is used together with Zoom or other configurations rooms that require dedicated audio/video controls. The options below power the Zoom keys of the section 4 · Media Window.

Zoom

  • Zoom controls: show the keys dedicated to Zoom in the Media Window.
  • Zoom Sharing: enables the button that sends the sharing shortcut to Zoom.
  • Show Zoom to the public: enables the button that temporarily hides the JW Time Public Screen to show the speakers or the Zoom window already positioned below, on the room monitor.
  • Sharing Shortcut- This must be the same global shortcut configured within Zoom to start or stop screen sharing.
  • Test: Send the shortcut and check if Zoom opens or changes the share status.

Zoom shortcut

JW Time does not choose the window or screen to share in Zoom for you: send the key combination that you indicated in this panel. For this reason the same shortcut must also be active in the settings of Zoom, preferably as a global shortcut. If you use a different combination in Zoom, the Sharing key and Zoom Auto mode may not work reliably.

Before the meeting, check the complete flow: prepare Zoom on the public monitor, place the Public Screen below the speakers window or the duplicate Zoom screen, try the shortcut with Test and then use the controls from the Media Window.

Media automation

  • Auto start: when the key Zoom Auto is active in the Media Window, starts sharing when a media is played.
  • Stop sharing when media closes: stops Zoom sharing when the media is closed or ends, only if sharing was started by JW Time.
  • Initial Pause: Prepares the media to pause after sharing starts.
  • Restore focus: Returns focus to the Media Window after the Zoom action.

Use with Video Conference Meetings

If someone needs to consult or control JW Time from offline during a video conference meeting, it cannot be configured from Zoom: use section 8.3 · Remote access for videoconference meetings. In this panel only shortcuts, sharing and management of the Zoom window remain.

15 · Connection and Server ↑ Table of Contents

This panel manages the app's connectivity: the Web Monitor and the Web Server for viewing and control from the browser, the QR for quick access, remote control, advanced connection tests and log management.

15.1 · Web Monitor and Web Server

  • JW Time - Web Monitor: activate or deactivate the internal server.
  • Server Address: shows the URL to open in the local network. In the manual we use a generic address, for example http://192.168.1.100:8000.
  • Copy address: Copies the URL to the clipboard.
  • QR: shows a QR code to open the Web Monitor more quickly from your phone or tablet.
  • Server Port: Set the HTTP port, normally 8000.
  • Editor / Controller: enable passwords to edit or control timer and program from the browser. With the Controller role, the Web Monitor also becomes an operating panel: it can start, pause, stop and update the available values without returning to the main computer.
  • Added the ability in the Web Monitor to edit song numbers and part names, in both Editor mode and Controller mode.

When the Web Server is active, JW Time can be opened from another device on the same local network via browser, to use the Web Monitor from your phone, tablet or another computer.

15.2 · Remote Web Server

This section is used to make the Web Monitor reachable even outside the local network, for example during meetings via videoconference. For context of use, see the section 8.3 · Remote access for videoconference meetings; here you will find the practical controls.

  • Generate remote link: creates a temporary link via Cloudflare component when access outside the local network is needed.
  • Download component: download the component necessary for the remote connection.
  • Remote Link: Shows the generated link and Copy/QR buttons.
  • Remote PIN: Protects access to remote functions when configured.
  • Link Status- Indicates whether the remote component, active connections, and Cloudflare are available.

15.3 · Remote Control

If the function is enabled, the Web Server can be used not only for viewing, but also for executing Remotely timer and program operations. When the Controller role is enabled, the web screen shows also controls to intervene on available values, for example the duration of a part.

Among these the following may be available:

  • start the timer
  • pause and resume
  • stop
  • control the program/schedule

Depending on the configuration, different access levels may be available, in order to distinguish simple consultation from the control or modification functions.

15.4 · Connection Test

  • Connection test: check network, JW.org, proxy and write permissions.
  • Copy result: copies the test result for assistance or diagnosis.
  • Open log folder: directly opens the app log folder.

16 · Recordings ↑ Table of Contents

Audio recording function of meetings, accessible from the "Recordings" tab in the settings and from the REC button in the main window.

Automatic recording

  • Automatic recording active: starts recording when the timer starts, if the function is enabled.
  • Pause recording when timer pauses: Also pauses recording during timer pauses.
  • Recording mode: choose between one file per part or a single file per session.

Path and devices

  • Path: indicates the folder in which to save the files. A generic path is shown in the manual.
  • Choose path: select the destination folder.
  • Open path: opens the recordings folder.
  • PC Audio: Select the computer audio source.
  • Microphone: Select the microphone to record along with the PC audio.
  • PC Volume / Mic Volume: adjusts the recorded level of the two sources.
  • PC Threshold: defines when PC audio should be considered active.
  • Mic Reduction.: Automatically lowers the microphone when PC audio exceeds the threshold.
  • Registration test: check devices and levels before the meeting.
  • Statistics: shows information on the recordings made.
