Quick Start (3 minutes)

  1. Choose the language

    Go to Settings → General → Language and select the desired language. This will automatically download the correct schedule (if available for that language) and set it for the current week.

  2. Select monitor and enable Timer Screen

    Still in General, choose the external monitor and activate Enable Timer Screen. (Optional): enable 12h format or activate adaptive Bible study and/or adaptive Watchtower Study. The app will automatically adjust the duration of these parts to help you meet the meeting closing time.

  3. Check the loaded schedule

    From the bottom bar open Program: it should already show Midweek or Weekend based on the day. (If you want to customize) choose Manual.

  4. Start and manage timers

    Press ▶ Start to begin. Use ❚❚ Pause during speaker changes and ▶▶ Next to move to the next item. Keep an eye on the Estimated End Time.

  5. Communicate with the speaker (if needed)

    Press Message and send full screen or partial screen text: it will appear on the Timer monitor.

  6. Close the meeting

    At the end press ⏹ Stop to close the schedule. If you use the Web Monitor, you can print the schedule or follow it from a browser on the same network.

Part I · Interface and Operation

1 · Main Screen ↑ Table of Contents

The main screen is the heart of the application. From here you can start, manage and monitor all meeting timers, view main information such as the current part title, the clock and remaining time, as well as control important messages and notifications.

Main areas

1.1 · Part Title and Clock

The top row shows the title of the current part, while below is displayed the clock with the current time or, when you start a timer, the countdown of the remaining time. In 12-hour format the AM/PM suffix also appears in reduced size to improve readability.

1.2 · Timer Table

Lists the meeting schedule (title, duration, status). Content loads automatically based on the selected day: the app recognizes if it's Midweek or Weekend and proposes the correct schedule. For buttons and detailed actions see 2. Timer Manipulation.

1.3 · Estimated End Time

Shows the estimated closing time based on the schedule. Next to it appears the deviation:
+m:ss (yellow within 1', red beyond) = delay
−m:ss (green) = early
The time and indicator update in real time. If adaptive Bible study / Watchtower are active, durations are recalculated to meet the closing time.

1.4 · Pause/speaker change time

Cumulative hh:mm:ss counter for technical pauses/speaker changes: it sums the time the timer was paused. While paused, the part timer stays stopped while this counter continues to increase. Click the indicator to reset the total.

1.5 · Main timer control buttons

Controls to start, stop and manage timers: Stop (red), Pause, and Start/Next (green). The Pause button pauses the part timer (the count stays stopped) while the pause/speaker change time continues to increase. To exit pause, press Pause again or move to the next part with Start/Next: the accumulated time is recorded in the counter described in 1.4.

1.6 · News (communications)

When there are official communications for the app, a "News" box/notice appears on the Main Screen. Clicking it opens the panel with the updates summary.

1.7 · Bottom Control Bar

Bar with buttons Settings, Minimize/Maximize, Program (week selection), Message and Online Manual. For more information see the dedicated chapters.

2 · Timer manipulation ↑ Table of Contents

This section illustrates the main timer management operations — how to add, modify or delete items — and the use of start, skip and reset commands.

2.1 · Main operations

  1. Add / Delete rows: click on + or – buttons.
  2. Edit Title/Duration: double click on cell, then Enter.
  3. Duration Formats: 5 → 5:00, 1:30 → 1:30 minutes.
  4. Start/Skip/Reset:
    • ▶ starts.
    • ▶▶ skips to next.
    • ↺ resets to original duration.

2.2 · Real-time updates

The timer updates constantly, synchronizing the Duration and Timer values in the table with the ongoing countdown, even if durations are modified or the part is changed.

2.3 · Automatic adjustment — Bible Study and Watchtower Study

If Adaptive Bible study and/or Adaptive Watchtower Study are active in Settings → General, the app dynamically adapts the duration of the selected part to maintain the meeting closing time. The adjustment only concerns parts with the active option and can automatically increase or reduce minutes based on accumulated advance/delay. The Estimated End Time and deviation update in real time. You can always manually intervene on timers: the adaptive recalculates starting from the new situation.

3 · Timer and meeting popup ↑ Table of Contents

The meeting start popup appears automatically at the set time with the chosen message, while the Timer Screen shows clock and countdown on an external monitor.

3.1 · Meeting Timer Popup

Appears at the set time and shows the custom message along with the countdown. Closes automatically when time expires.

  • Shows the custom text configured in settings
  • Countdown visible until meeting start
  • Automatic closure when expired

3.2 · External Timer Screen

When you activate "Enable Timer Screen" from settings, you get a full screen display on an external monitor, ideal for the podium or chairman.

  • Full screen display on selected monitor
  • Shows clock or countdown of current part
  • Tips mode: shows the time of the previous part to provide immediate feedback

4 · Weekly navigation ↑ Table of Contents

This section illustrates how to load the correct schedule for the week, create a manual schedule when needed and quickly scroll through previous, current and next weeks.

By default, the app automatically loads the current week's schedule: midweek from Monday to Friday and weekend on Saturday and Sunday.

4.1 · Automatic scheduling

  1. Midweek: loads the weekday schedule.
  2. Weekend: loads the Saturday/Sunday schedule, asking for the public talk song.

4.2 · Week selector and calendar preview

The new Week button always shows the range (e.g. "Week: Oct 20 – 26") with a badge indicating the offset from the current week (0, +1, -2, etc.). A click opens the compact calendar to jump directly to the desired week without scrolling the Previous/Next buttons multiple times.

  • Immediate badge: you understand if you're looking at the current, future or past week.
  • Weekly calendar: choose weeks directly in a practical way.

4.3 · CO Visit

The CO Visit button, placed between the Week selector and Load from backup, transforms the schedule when there's a circuit overseer visit. When pressed the button lights up (ON) and the app automatically prepares the special parts planned for the visit.

  • Midweek meeting: the Congregation Bible Study is converted to a fixed 30-minute part with title "CO Visit".
  • Weekend: a dedicated item is added after the Watchtower Study and durations adapted to meet the closing time.
  • Dedicated colors and popups: the CO Visit row adopts its own palette and activates the specific closing song popup.
  • Quick deactivation: turning off the toggle, the table immediately returns to the standard schedule without losing other changes.

4.4 · Load from backup

The Load from backup button opens a list of locally saved packages, sorted by date, meeting type and language. Ideal for working offline or restoring programs already prepared on another PC.

  • Detailed list: each row shows week start date, type (Midweek/Weekend) and language.
  • Safe loading: packages are verified and applied directly to the current table.
  • Clear messages: if no backups are available, the app immediately shows a notice.

4.5 · Manual List Mode

Allows you to create a completely custom list, manually setting titles and durations according to your needs.

  • You can save the list to reuse it whenever you want.
  • You can recall it later via "Load manual list", without having to rewrite it every time.

5 · Message management ↑ Table of Contents

The Messages function allows you to communicate discreetly and clearly with the speaker during the meeting, allowing you to notify or give directions without interrupting or disturbing the part. Messages appear on the timer screen, so the speaker sees them in real time without distractions.

5.1 · Full screen message

  1. Click in the area where it says Insert message.
  2. Enter the text.
  3. Press Full screen → the text appears full screen on the Timer monitor.

5.2 · Partial screen message

  1. Click in the area where it says Insert message.
  2. Enter the text.
  3. Press Partial screen → the text appears at the top of the Timer window, allowing the speaker to see both the message and their available time.

5.3 · Timer Mirror

When a message is sent, a resizable and movable preview window automatically appears showing exactly what the speaker sees on the Timer monitor. This allows you to verify in real time the display of the message and timer.

6 · REC button and recording panel ↑ Table of Contents

The REC button is located in the bottom bar next to Program and Message. The dot next to the text shows the recording status (waiting, active or MP3 conversion), while the linked menu and toggle area provide quick controls to open the JWTimeREC folder, manage manual recordings and monitor PC and microphone audio levels.

6.1 · Button indicators and states

  • Waiting: the dot remains gray and the menu shows "Waiting" / "No active recording".
  • Recording in progress: the dot becomes flashing red and the menu shows "Recording in progress" with part title and timer.
  • MP3 conversion: the dot turns orange and the button remains disabled until file export completes.

6.2 · Button quick menu

The menu that opens by clicking REC allows quick controls:

  • Open folder: directly opens the JWTimeREC directory containing recordings.
  • Enable/Disable recording: enables or stops manual recording without entering settings.
  • Inline details: under the status appear the active row title and updated duration.
  • Automatic messages: immediately indicates when recording is disabled or when a conversion is in progress.

6.3 · Toggle area with audio levels

Docking the REC button to the toggle area shows an always visible panel with the same information as the menu but enriched with audio meters. Here you can start/stop manual recordings, monitor PC/Mic levels and open the JWTimeREC folder with a single click.

7 · Web Monitor & Server ↑ Table of Contents

This is the web page viewable when connected to the same network as the application. It allows you to show the timer remotely on devices such as projectors, tablets or external computers — useful for the chairman monitoring the timer of the various parts. This way you can monitor the meeting progress in real time and print the schedule directly from the browser.

For Web Monitor and server configuration, see 10. Server in the settings panels.

From the browser you can:

  • View timer and countdown full screen.
  • Print the schedule.
  • Check time progress in real time.

Part II · Settings Panels

In this part you can configure the app's behavior: from general settings (language, 12/24h format and Timer Screen), to meeting scheduling (popup and dedicated monitor), to Web Monitor & Server and Backup & Restore.

The SAVE button applies all changes.

8 · General ↑ Table of Contents

  • Select language: changes the interface language. When changing language, the app automatically updates the weekly program in the new language (if available).
  • Select monitor: chooses the external screen on which to show the Timer.
  • Enable Timer Screen: displays clock or countdown full screen on the selected monitor.
  • Adaptive Bible study: when activated, the Congregation Bible Study duration is automatically adjusted to help meet the closing time.
  • Adaptive Watchtower Study: when activated, the Watchtower Study duration is automatically adjusted to help meet the closing time.
  • Use 12h format (AM/PM): shows time in 12-hour format. If disabled, uses 24h format.

9 · Meetings (Meeting Start Timer) ↑ Table of Contents

This panel is used to schedule the meeting start countdown at the set time. You set day, time and advance (in minutes) with which you want to start the countdown.

9.1 · Scheduling (Day · Time · Advance)

  • Day: choose the day of the week when the meeting is held.
  • Time: set the meeting start time.
  • Advance (min): how many minutes before the set time the countdown should appear on the timer screen.

9.2 · Meeting Start Message

Write the text that will appear on the screen together with the countdown (e.g. "Welcome — the meeting starts soon"). Keep it short and readable. (see point 3.1)

Note: this function does not change the meeting duration and does not affect "Estimated End Time".

10 · Connection and Server ↑ Table of Contents

This panel manages the app's connectivity: from the Web Monitor to display the timer via browser, to advanced connection tests and log management.

10.1 · Web Monitor and Server

  • WEB Monitor: enables or disables the internal server.
  • Server Address: shows the URL and includes the Copy address button to copy it automatically.
  • Port: sets the HTTP port (default: 8000).
  • Light/dark theme: the web server page now supports both themes.

10.2 · Connection Test

  • Verify internet connection and necessary sites
  • HTTPS and certificate check
  • System proxy verification
  • Time synchronization check
  • Folder write test (log, cache, backup)

10.3 · Log Management

Direct opening of log folder from settings with clearer and more useful messages for debugging.

11 · Recordings ↑ Table of Contents

Meeting audio recording function, accessible from the "Recordings" tab in settings and from the REC button in the main window.

Main features

  • Automatic recording: starts recording autonomously at the beginning of each meeting, following the set schedule.
  • Destination folder choice: select where to save recordings
  • Recording test: verify proper operation before the meeting
  • Volume adjustment: microphone and PC audio adjustable separately
  • Automatic "ducking" function: automatically lowers Mic volume when videos are projected
  • Space and file statistics: monitor space used by recordings

12 · Backup & Restore ↑ Table of Contents

This panel allows you to manage backups of settings and weekly programs, both for normal use and for OFFLINE mode.

12.1 · Settings Backup

  • Save Backup: exports configurations to a .jwtimesettings file (new version 2.0 format).
  • Load Backup: imports a configuration file. Also supports old .json files. After import, press Save to make changes effective.

12.2 · OFFLINE Backup

Automatic backup of weekly programs:

  • Up to 12 weeks stored automatically
  • Program recovery even without internet connection
  • Works in background without manual intervention

Exported backups for OFFLINE use:

  • Use the "Download" button to get the file immediately
  • Transfer files to other PCs even without internet connection
  • Perfect for computers in the hall not connected to the network
  • Supports multilingual management with separate backups for each language
