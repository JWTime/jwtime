Media and meeting preparation NEW

The Media Window offers a support tool for preparing and checking available content more easily.

Note: the Media Window is a technical support feature. It helps prepare and check available content more easily, but its practical use always remains tied to local needs, congregation direction, and arrangements.

  • Add local files, images, videos, PDFs and URLs.
  • View songs, official media and content linked to the week.
  • Organize manual content into groups and reorder items according to the program.
  • Check in advance what is available and what is missing.
  • Open or show content on the Public Screen without searching for it in different folders.

Official media, playlists and packages NEW

JW Time can help organize and reuse official content that is already available and local content in a more orderly way, even when you need to work offline.

  • Download and reuse official media already present locally.
  • Manage a dedicated cache for official songs, with availability checks.

Important note: the alternate path introduced in JW Time 2.1.0 applies to weekly and weekend programs. Official media do not have a second recovery server. If a router, firewall, DNS service, or content filter blocks JW Time from reaching jw.org or religious sites, the Media Window may not be able to download official media. Time Counter Screen and Public Screen NEW

The displays have been separated: the Time Counter Screen is for the conductor, the Public Screen is for the room.

  • Start from presets ready for the Time Counter Screen.
  • Edit layout, layers, colors, backgrounds, logo, clock and texts with the editor.
  • Show images, PDFs, videos and visual content on the Public Screen.

Version 2.1.0

SOLUTION FOR ALL CONGREGATIONS THAT ARE UNABLE TO DOWNLOAD THE PROGRAMS.

Programs always reachable, with new download options.

⚠️ Problems with the Web Server?

WARNING: Before proceeding, make a backup of the app as you will need to uninstall it.

Some users upgrading from previous versions may experience issues with the Web Serverfeature. If the Web Server is unresponsive, follow these steps to clean the firewall rules:

Solution (one-time use):

  1. Press Start and search for: "Windows Defender Firewall with Advanced Security". Open it.
  2. On the left click on Incoming connection rules.
  3. In the list, look for entries that contain JWTime.
  4. Delete them all (right click on the rule → Delete).

After deleting the rules, reinstall JWTime and reload the settings. Then try using the WEB Monitor.
Once you have done this operation, you will no longer need to do it again.

Version 1.8.4

Major Update

Adaptive Watchtower Study

The app automatically adjusts the duration of the Watchtower Study to help close at the scheduled time of the meeting. This function is activated from the General Settings.

Time Offset Indicator

Real-time reporting system that shows if the meeting is:

  • Late (Red): Over 1 minute late
  • Late (Yellow): Up to 1 minute late
  • Ahead (Green): Meeting proceeds faster than expected
  • Right on time: No color displayed

The indicator automatically updates in real time during the meeting.

Technical Improvements

  • Optimized timer synchronization for greater accuracy
  • Improved stability of the Time Counter Screen on external monitors
  • Minor graphical fixes to the interface
  • Optimized overall performance
