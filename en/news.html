⚠️ Problems with the Web Server?

WARNING: Before proceeding, back up the app because it will need to be uninstalled.

Some users updating from previous versions may experience issues with the Web Server feature. If the Web Server does not respond, follow these steps to clear the firewall rules:

Solution (run only once):

  1. Press Start and search for "Windows Defender Firewall with Advanced Security". Open it.
  2. On the left, click Inbound Rules.
  3. In the list, look for entries that contain JWTime.
  4. Delete them all (right-click the rule → Delete).

After deleting the rules, reinstall JWTime and reload the settings. Then try using the Web Monitor.
Once this operation is done, you will not need to repeat it.

⚠️ Backup Recommendation

This version adds many new options that are not present in old backups: the new save format is .jwtimesettings, but you can still load old backup files in .json format. After configuring the new version 2.0 settings, it is recommended to also create a new backup in .jwtimesettings format, so all new options will also be saved.

Version 1.8.4

Important Update

Adaptive Watchtower Study

The app automatically adjusts the Watchtower Study duration to help close at the scheduled meeting time. This function is activated from General Settings.

Time Deviation Indicator

Real-time alert system showing if the meeting is:

  • Running late (Red): More than 1 minute late
  • Running late (Yellow): Up to 1 minute late
  • Running early (Green): The meeting is progressing faster than expected
  • Perfectly on time: No color displayed

The indicator updates automatically in real-time during the meeting.

Technical Improvements

  • Timer synchronization optimized for greater precision
  • Improved stability of Timer Screen on external monitors
  • Minor graphical corrections to interface
  • Overall performance optimized
