What's New

Discover the latest features and improvements in JW Time

Version 3.0.0 coming soon

Customizable Timer Display, added Congregation management, remote control through the Web Server, and a completely redesigned interface.

With version 3.0.0, JW Time takes an important step forward.
The main new feature is the customizable Timer Display, which lets you better adapt the view to your needs, starting from ready-made presets and adjusting them easily.

Alongside this evolution, the app also introduces added Congregation management, more flexible recording, and a simpler initial setup, in addition to a completely redesigned interface and a more useful Web Server that offers faster browser access and remote control for several app operations.

Preview of the new interface (3.0 screenshot in progress).

New customizable Timer Display NEW

The most important new feature in JW Time 3.0.0 is the new customizable Timer Display system.

The Timer Display is no longer limited to a fixed layout. You can adapt it to your needs and get a cleaner, more readable screen that fits your local use case.

To make this feature easy from the first use, JW Time also introduces starter presets that let you begin with a ready-to-use layout. You do not need to start from scratch: choose a suitable base, keep it as is, or customize it further only if you need to.

This approach makes initial setup easier and helps you get a good result in just a few steps.

Added Congregation management NEW

Congregation management has been expanded to make it easier to work in different contexts without reconfiguring the app from zero each time.

In practice, you can organize settings, preferences, and congregation-related data more clearly, keeping everything easier to manage over time.

This is especially useful when multiple people use the app or when you need to switch quickly between different configurations.

Manage timers and program remotely NEW

JW Time 3.0.0 makes the Web Server even more useful: in addition to browser viewing, it now allows remote control of several timer and program operations.

This makes practical use from another device on the same local network, especially when quick intervention is needed without using the main computer directly.

Access is also faster thanks to QR, which opens the Web Server quickly from phone or tablet. Where available, remote control can be protected with the provided access options.

Completely redesigned interface NEW

JW Time has also been updated visually, with a more modern, cleaner, and more consistent interface.

The new visual style improves overall clarity and makes it easier to read the different areas, both in the main window and in configuration screens.

The result is a more polished, more current, and more uniform experience across the entire application.

More advanced settings and clearer setup IMPROVED

App configuration has also been made more organized and more useful in day-to-day use.

General settings now provide clearer access to key options, including Timer Display layout management and settings related to the new customization experience.

This makes it easier to configure JW Time from the start, especially for users who want to begin with ready-made solutions and edit only what is really needed.

More flexible recording IMPROVED

Recording has been made more flexible to better fit real meeting flow, where quick adjustments are often needed during different phases.

The goal is to reduce operational interruptions: you can handle pause, resume, and overall flow more easily without losing control.

For users, this means a more convenient and predictable workflow, with fewer manual steps when focus is needed.

Other general improvements IMPROVED

In addition to the main new features, JW Time 3.0.0 also includes several overall improvements that make the app more coherent, cleaner, and easier to use.

Many of these refinements improve overall experience quality, with better visual consistency across screens and clearer setup in multiple areas of the app.

⚠️ Problems with the Web Server?

WARNING: Before proceeding, back up the app because it will need to be uninstalled.

Some users updating from previous versions may experience issues with the Web Server feature. If the Web Server does not respond, follow these steps to clear the firewall rules:

Solution (run only once):

  1. Press Start and search for "Windows Defender Firewall with Advanced Security". Open it.
  2. On the left, click Inbound Rules.
  3. In the list, look for entries that contain JWTime.
  4. Delete them all (right-click the rule → Delete).

After deleting the rules, reinstall JWTime and reload the settings. Then try using the Web Monitor.
Once this operation is done, you will not need to repeat it.

Version 1.8.4

Important Update

Adaptive Watchtower Study

The app automatically adjusts the Watchtower Study duration to help close at the scheduled meeting time. This function is activated from General Settings.

Time Deviation Indicator

Real-time alert system showing if the meeting is:

  • Running late (Red): More than 1 minute late
  • Running late (Yellow): Up to 1 minute late
  • Running early (Green): The meeting is progressing faster than expected
  • Perfectly on time: No color displayed

The indicator updates automatically in real-time during the meeting.

Technical Improvements

  • Timer synchronization optimized for greater precision
  • Improved stability of Timer Screen on external monitors
  • Minor graphical corrections to interface
  • Overall performance optimized

You can:

  • Reset: Reset accumulated technical time

Adaptive Bible Study

First version of the adaptive function, applied to the Congregation Bible Study. The app automatically adjusts the duration to maintain the expected closing time.
