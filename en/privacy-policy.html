Privacy Policy

How JW Time handles local data, connections and diagnostics

Last update: 20 June 2026

1. Introduction

JW Time is an independent, free, non-commercial project. This policy explains what data is stored locally, what Internet connections may be made, and what controls the user has.

JW Time does not automatically collect or send personal data, telemetry, usage reports, crash reports or logs to the developer.

2. Data Stored Locally

JW Time may store app settings, congregation profiles, programs, imported files, media cache, program cache, audio recordings if enabled, backups and technical logs on the computer.

This data is used to run the app, keep preferences, work offline and help the user diagnose local problems.

3. Data Not Sent Automatically

JW Time does not automatically send personal data, usage statistics, telemetry, crash reports, logs or connection test results to the developer.

If you request support, you decide whether to manually copy and send logs, screenshots or diagnostic results. In that case, the shared data is used only to provide support.

4. Internet Connections

JW Time may connect to official sources to download or update public programs and media, to the JW Time site for documentation, communications and support files, and to external services only for explicit features such as network diagnostics or temporary remote access.

These requests are used to retrieve content or check connectivity. They are not telemetry sent to the developer.

5. Local Web Server and Remote Access

If enabled, the JW Time Web Server publishes monitoring and control functions on the local network. Devices on the same LAN can see operational data such as part title, remaining time and timer status.

If the user enables temporary remote access, traffic passes through the external service needed to create that link. The user controls activation and available protections, such as PINs or roles.

6. Retention and Deletion

Data remains on the user’s computer. Some temporary files, cache files or old packages are cleaned automatically where supported by the app; others can be deleted manually from settings or by removing the app’s local folders.

Local files added manually remain where the user chose to keep them and are not removed by JW Time without an explicit action.

7. User Rights

Because JW Time does not maintain a remote archive of user data, access, correction and deletion are mainly performed on your own computer: from app settings, local folders, or by uninstalling/removing local data.

For practical details about content, media and cache, see the FAQ.

For assistance, you can contact us.
