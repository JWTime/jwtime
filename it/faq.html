FAQ

Risposte rapide su privacy, contenuti, cache e connessioni

Stato del progetto

JW Time è un’app ufficiale?

No. JW Time è un progetto indipendente, gratuito e non commerciale, creato da volontari. Non è affiliato, sponsorizzato o approvato ufficialmente da alcuna organizzazione.

Il nome JW Time indica che l’app è ufficiale?

No. Il nome descrive l’uso previsto dell’app, cioè aiutare a gestire tempi e supporto tecnico. Non vuole comunicare approvazione ufficiale o rappresentanza da parte di un’organizzazione.

JW Time è commerciale?

No. JW Time è distribuito gratuitamente e non contiene pubblicità, acquisti in-app o funzioni a pagamento.

Contenuti e media

JW Time rispetta le indicazioni ufficiali?

JW Time è progettato per un uso locale, personale, gratuito e non commerciale, rispettando le indicazioni ufficiali sull’uso dei contenuti. L’app non include contenuti ufficiali nell’installer e non li ridistribuisce tramite server propri.

JW Time fa scraping live dei siti ufficiali?

No. JW Time non esplora né copia pagine web in tempo reale. Per programmi e media usa file pubblici e canali di distribuzione ufficiali; i contenuti scaricati vengono preparati e usati localmente sul computer dell’utente.

Come vengono recuperati i media?

Quando usi la finestra Media, JW Time può richiedere i file tramite canali ufficiali disponibili pubblicamente e salvarli in cache locale. In alternativa puoi aggiungere file locali già presenti sul computer.

Privacy e connessioni

JW Time raccoglie dati personali o telemetria?

No. JW Time non raccoglie né invia automaticamente allo sviluppatore dati personali, telemetria, report d’uso, crash report o log. Impostazioni, profili, programmi, registrazioni, log e cache restano sul computer dell’utente.

Il test connessione invia dati allo sviluppatore?

No. Il test connessione mostra l’esito nell’app. Puoi copiarlo negli appunti e condividerlo manualmente solo se desideri assistenza.

Quali connessioni Internet può effettuare JW Time?

JW Time può collegarsi a sorgenti ufficiali per programmi e media, al sito JW Time per guida, comunicazioni e file di supporto, e a servizi esterni solo per funzioni esplicite come diagnostica di rete o accesso remoto temporaneo, se abilitate dall’utente.

File locali e cache

I file scaricati restano per sempre?

No. I file scaricati servono come cache locale o pacchetti offline. Alcuni file temporanei o vecchi vengono puliti automaticamente dove previsto; inoltre puoi svuotare cache, pacchetti e media offline dalle impostazioni dell’app. I file locali aggiunti manualmente restano nella posizione scelta dall’utente.

Posso usare JW Time offline?

Sì, per i contenuti già scaricati, importati o aggiunti manualmente. Il primo download e gli aggiornamenti dei contenuti richiedono una connessione Internet.
