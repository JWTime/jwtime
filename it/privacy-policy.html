Come JW Time gestisce dati locali, connessioni e diagnostica

Ultimo aggiornamento: 20 giugno 2026

1. Introduzione

JW Time è un progetto indipendente, gratuito e non commerciale. Questa policy spiega quali dati vengono salvati localmente, quali connessioni Internet possono essere effettuate e quali controlli ha l’utente.

JW Time non raccoglie né invia automaticamente allo sviluppatore dati personali, telemetria, report d’uso, crash report o log.

2. Dati salvati localmente

JW Time può salvare sul computer impostazioni dell’app, profili congregazione, programmi, file importati, cache media, cache programmi, registrazioni audio se abilitate, backup e log tecnici.

Questi dati sono usati per far funzionare l’app, mantenere le preferenze, lavorare offline e aiutare l’utente a diagnosticare problemi locali.

3. Dati non inviati automaticamente

JW Time non invia automaticamente allo sviluppatore dati personali, statistiche di utilizzo, telemetria, crash report, log o risultati dei test connessione.

Se richiedi assistenza, puoi decidere tu se copiare e inviare manualmente log, screenshot o risultati diagnostici. In quel caso i dati condivisi vengono usati solo per fornire supporto.

4. Connessioni Internet

JW Time può collegarsi a sorgenti ufficiali per scaricare o aggiornare programmi e media pubblici, al sito JW Time per documentazione, comunicazioni e file di supporto, e a servizi esterni solo per funzioni esplicite come diagnostica di rete o accesso remoto temporaneo.

Queste richieste servono a recuperare contenuti o verificare la connettività. Non sono telemetria inviata allo sviluppatore.

5. Server Web locale e accesso remoto

Il Server Web di JW Time, se attivato, pubblica funzioni di monitoraggio e controllo sulla rete locale. I dispositivi della stessa LAN possono vedere dati operativi come titolo della parte, tempo rimanente e stato del timer.

Se l’utente abilita un accesso remoto temporaneo, il traffico passa attraverso il servizio esterno necessario a creare il collegamento. L’utente controlla l’attivazione e le protezioni disponibili, come PIN o ruoli.

6. Conservazione e cancellazione

I dati rimangono sul computer dell’utente. Alcuni file temporanei, cache o pacchetti vecchi vengono puliti automaticamente dove previsto dall’app; altri possono essere eliminati manualmente dalle impostazioni o cancellando le cartelle locali dell’app.

I file locali aggiunti manualmente restano nella posizione scelta dall’utente e non vengono rimossi da JW Time senza un’azione esplicita.

7. Diritti dell’utente

Poiché JW Time non mantiene un archivio remoto dei dati dell’utente, accesso, correzione e cancellazione si esercitano principalmente sul proprio computer: dalle impostazioni dell’app, dalle cartelle locali o disinstallando/rimuovendo i dati locali.

Per dettagli pratici su contenuti, media e cache, consulta anche la FAQ.

Per assistenza puoi contattarci.
