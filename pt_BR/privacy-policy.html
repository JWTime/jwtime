🔒 Política de Privacidade

Como o JW Time gerencia e protege seus dados

Última atualização: 3 de junho de 2025

1. Introdução

O JW Time gerencia e exibe cronômetros durante reuniões ou montagens, sincronizando dados em redes locais, mantendo a conformidade de privacidade com os padrões GDPR quando aplicável.

2. Coleta de Dados Pessoais

Dados Fornecidos pelo Usuário

O aplicativo não exige a inserção de informações pessoais como nome, endereço de e-mail, número de telefone ou outros identificadores de informações.

Endereços IP e dados técnicos

O aplicativo pode registrar os endereços IP de dispositivos na rede local para fins depuração. De acordo com o GDPR, os endereços IP podem constituir dados pessoais quando vinculado a outras informações de identificação.

No entanto, JW Time apenas registra esses dados localmente e não os transmite nem os compartilha com servidores de terceiros.

Servidor web Integrado

Um servidor Web local exibe o status dos temporizadores para dispositivos na rede. O A transmissão de dados ocorre apenas dentro da LAN, limitada a títulos das peças e o tempo restante.

Nenhum dado pessoal sobre você é coletado, armazenado ou transmitido.

3. Finalidade e base jurídica

O processamento de dados baseia-se no interesse legítimo nos termos do artigo 6(1)(f) do GDPR para:

  • Funções de depuração e diagnóstico: Resolver problemas técnicos e melhorar o aplicativo
  • Monitorar a segurança da rede local: Verificar conexões com a rede local
  • Manter a funcionalidade do aplicativo: Garantir o funcionamento correto

4. Compartilhamento e transferência de dados

A JW Time NÃO envia informações para serviços externos, servidores em nuvem ou terceiros.

Todos os dados e registros do temporizador permanecem no dispositivo do usuário ou na rede local. Não há transferência de dados para a Internet.

5. Segurança e armazenamento de dados

Os logs são armazenados localmente no dispositivo do usuário (por exemplo, na pasta %APPDATA%\JWTimeLogs no Windows).

Os dados permanecem armazenados até que o usuário os exclua manualmente. Não há arquivo remoto ou backup automático em servidores externos.

Como excluir dados

  • Acesse a pasta de logs no seu dispositivo
  • Exclua arquivos manualmente
  • Ou desinstale completamente o aplicativo

6. Direitos do usuário sob GDPR

Se você é residente da União Europeia, você tem os seguintes direitos:

📋 Acesso

Direito de acessar dados registrados localmente no seu dispositivo

✏️ Retificação

Direito de corrigir ou atualizar os dados armazenados

🗑️ Exclusão

Direito de excluir logs e dados armazenados a qualquer momento

⛔ Limitação

Direito de limitar o processamento de seus dados

🚫 Objeto

Direito de oposição ao tratamento dos seus dados

⚖️ Reclamação

Direito de apresentar reclamação às autoridades de proteção de dados

7. Como exercer os seus direitos

Como todos os dados são armazenados localmente no seu dispositivo, você pode exercitar seus direitos diretamente:

  • Acesso: Navegue até a pasta de logs no seu dispositivo
  • Exclusão: Exclua arquivos de log da pasta %APPDATA%\JWTimeLogs
  • Restrição: Desative os recursos de registro nas configurações do aplicativo (se disponível)

Para obter mais assistência, você pode entrar em contato conosco.

8. Mudanças nesta Política

Esta Política de Privacidade pode ser atualizada periodicamente para refletir desenvolvimentos técnico ou regulatório. Os utilizadores são encorajados a consultar esta página regularmente.
