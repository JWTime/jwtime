🔒 Política de Privacidade

Como o JW Time gerencia e protege seus dados

Última atualização: 3 de junho de 2025

1. Introdução

O JW Time gerencia e exibe temporizadores durante reuniões ou assembleias, sincronizando dados através de redes locais mantendo a conformidade de privacidade de acordo com os padrões GDPR onde aplicável.

2. Coleta de Dados Pessoais

Dados Fornecidos pelo Usuário

O aplicativo não requer a inserção de informações pessoais como nome, endereço de e-mail, número de telefone ou outras informações de identificação.

Endereços IP e Dados Técnicos

O aplicativo pode registrar os endereços IP dos dispositivos na rede local para fins de depuração. De acordo com o GDPR, os endereços IP podem constituir dados pessoais quando vinculados a outras informações de identificação.

No entanto, o JW Time registra esses dados apenas localmente e não os transmite nem os compartilha com servidores de terceiros.

Servidor Web Integrado

Um servidor web local exibe o status dos temporizadores para dispositivos na rede. A transmissão de dados ocorre apenas dentro da LAN, limitada aos títulos das partes e ao tempo restante.

Nenhum dado pessoal do usuário é coletado, armazenado ou transmitido.

3. Propósito e Base Legal

O processamento de dados é baseado no interesse legítimo nos termos do Artigo 6(1)(f) do GDPR para:

  • Funções de depuração e diagnóstico: Resolver problemas técnicos e melhorar o aplicativo
  • Monitoramento de segurança da rede local: Verificar conexões à rede local
  • Manutenção da funcionalidade do aplicativo: Garantir o funcionamento adequado

4. Compartilhamento e Transferência de Dados

O JW Time NÃO envia informações para serviços externos, servidores na nuvem ou terceiros.

Todos os dados de temporizador e registros permanecem no dispositivo do usuário ou na rede local. Não há transferência de dados para a Internet.

5. Segurança e Retenção de Dados

Os registros são armazenados localmente no dispositivo do usuário (por exemplo, na pasta %APPDATA%\JWTimeLogs no Windows).

Os dados permanecem armazenados até que o usuário os exclua manualmente. Não há armazenamento remoto ou backup automático em servidores externos.

Como Excluir os Dados

  • Acesse a pasta de registros no seu dispositivo
  • Exclua os arquivos manualmente
  • Ou desinstale completamente o aplicativo

6. Direitos do Usuário segundo o GDPR

Se você reside na União Europeia, você tem os seguintes direitos:

📋 Acesso

Direito de acessar dados registrados localmente no seu dispositivo

✏️ Retificação

Direito de corrigir ou atualizar dados armazenados

🗑️ Exclusão

Direito de excluir registros e dados armazenados a qualquer momento

⛔ Limitação

Direito de limitar o processamento dos seus dados

🚫 Oposição

Direito de se opor ao processamento dos seus dados

⚖️ Reclamação

Direito de apresentar uma reclamação às autoridades de proteção de dados

7. Como Exercer Seus Direitos

Como todos os dados são armazenados localmente no seu dispositivo, você pode exercer seus direitos diretamente:

  • Acesso: Navegue até a pasta de registros no seu dispositivo
  • Exclusão: Exclua os arquivos de registro da pasta %APPDATA%\JWTimeLogs
  • Limitação: Desative os recursos de registro nas configurações do aplicativo (se disponível)

Para obter assistência adicional, você pode entrar em contato conosco.

8. Mudanças Nesta Política

Esta Política de Privacidade pode ser atualizada periodicamente para refletir desenvolvimentos técnicos ou regulatórios. Os usuários são incentivados a consultar esta página regularmente.
