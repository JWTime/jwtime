Novidades

Descubra as últimas funcionalidades e melhorias do JW Time

Versão 2.0.0

Uma versão completamente renovada com funcionalidades avançadas para gerenciamento de reuniões.

⚠️ Recomendação de Backup

Com esta versão são adicionadas muitas opções novas que não estão presentes nos backups antigos: o novo formato de salvamento é .jwtimesettings, mas você ainda pode carregar arquivos de backup antigos no formato .json. Após configurar as novas opções da versão 2.0, é recomendável criar também um novo backup no formato .jwtimesettings, assim todas as novas opções também serão salvas.

Idioma Português Adicionado (Brasil) NOVO

O idioma Português (Brasil) foi adicionado, selecionável no menu Configurações.

Gravação de Reuniões NOVO

A versão 2.0 introduz uma nova função de gravação de áudio das reuniões.

Onde se encontra:

  • Janela Configurações: nova aba "Gravação"
  • Janela principal: novo botão REC

Funcionalidades principais:

  • Escolha da pasta de destino
  • Teste de gravação
  • Ajuste de volumes de microfone e áudio do PC
  • Função "ducking" automática
  • Estatísticas de espaço e arquivos
  • Controle manual com botão REC

Backup para Modo OFFLINE NOVO

Backup automático dos programas semanais:

  • Até 12 semanas armazenadas automaticamente
  • Recuperação de programa mesmo sem conexão
  • Funciona em segundo plano

Backups exportados para uso OFFLINE:

  • Exportação manual de pacotes de programas
  • Transferíveis para outros PCs sem internet
  • Ideal para PCs no Salão do Reino não conectados à rede

Gerenciamento multilíngue:

  • Backups distintos para cada idioma (IT, EN, ES, DE, pt_BR)
  • Sem sobreposição entre diferentes idiomas

Conexão, Servidor Web e Log MELHORADO

Teste de conexão avançado:

  • Verificar conexão à internet e sites necessários
  • Verificar HTTPS e certificados
  • Verificar proxy do sistema
  • Verificar sincronização de horário
  • Testar gravação de pastas (log, cache, backup)

Logs melhorados:

  • Abertura direta da pasta de log nas configurações
  • Mensagens mais claras e úteis

Página do Servidor web:

  • Suporte a tema claro/escuro

Salvar Listas Manuais NOVO

Quando você pressiona o botão "Manual" na aba Programas, agora você tem dois novos botões:

  • "Salvar lista manual" - Salva sua lista personalizada
  • "Carregar lista manual" - Recupera uma lista salva

Vantagens:

  • Criar listas completamente personalizadas
  • Salvar com um nome identificativo
  • Recuperar sem ter que reescrever
  • Os botões desaparecem quando você retorna para Meio de semana/Fim de semana

Novo Tema Claro MELHORADO

O novo tema claro foi introduzido, completamente adicionado na versão 2.0.

Melhorias do tema claro:

  • Cores mais legíveis para Nossa Vida Cristã e Faça Seu Melhor no Ministério
  • Seleção de células mais evidente
  • Cores de texto corrigidas na tabela e exibição ativa
  • Corrigido defeito no popup de cântico do orador

Interface geral:

  • Melhorias para maior clareza
  • Design mais uniforme e coerente

Versão 1.8.4

Atualização Importante

Estudo de A Sentinela Adaptativo

O aplicativo ajusta automaticamente a duração do Estudo de A Sentinela para ajudar a encerrar no horário programado da reunião. Esta função é ativada nas Configurações Gerais.

Indicador de Desvio de Horário

Sistema de sinalização em tempo real que mostra se a reunião está:

  • Atrasada (Vermelho): Mais de 1 minuto de atraso
  • Atrasada (Amarelo): Até 1 minuto de atraso
  • Adiantada (Verde): A reunião está progredindo mais rápido do que o previsto
  • Perfeitamente no horário: Nenhuma cor exibida

O indicador atualiza automaticamente em tempo real durante a reunião.

Melhorias Técnicas

  • Sincronização de temporizadores otimizada para maior precisão
  • Estabilidade melhorada da Tela de Cronômetro em monitores externos
  • Correções gráficas menores na interface
  • Desempenho geral otimizado

Versão 1.8.0

Funcionalidade Pausa

Função Pausa

Introduzida a função Pausa (ZZ) que pausa o temporizador atual mostrando o indicador ⏸. O aplicativo acompanha o "Tempo de pausa/troca de orador total".

É possível:

  • Retomar: Continuar de onde parou
  • Zerar: Reiniciar o tempo técnico acumulado

Estudo Bíblico Adaptativo

Primeira versão da função adaptativa, aplicada ao Estudo Bíblico de Congregação. O aplicativo ajusta automaticamente a duração para manter o horário de encerramento previsto.

Correções de Bugs

  • Resolvidos problemas de sincronização entre tela principal e monitor externo
  • Melhorada a estabilidade geral da aplicação
